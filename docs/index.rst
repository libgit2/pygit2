######################################################################
pygit2 - libgit2 bindings in Python
######################################################################

Bindings to the libgit2 shared library, implements Git plumbing.
Supports Python 3.6+ and PyPy3 7.3+

=====================================

- Documentation - http://www.pygit2.org/
